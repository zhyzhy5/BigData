
## 一.并发修改异常
	1)出现的现象：迭代器遍历集合，集合修改集合元素
	2)出现的原因：迭代器依赖于集合，而集合的改变迭代器并不知道
	3)解决方案：
		3.1)迭代器遍历，迭代器修改（ListIterator）
		3.2)集合遍历，集合修改（size()和get()），元素添加在集合的末尾


## 二.List、Set、Map和Queue之间的区别和使用场景
* List
	单列接口，继承Collection接口，该容器中的元素有序、可重复
	每一个元素都有索引值

* Set
	单列接口，继承Collection接口，该容器中的元素无序、唯一

* Map
	key-value接口，key列的数据结构是set，无序唯一，value列的数据结构是List，
	有序可重复，
	
* Queue
	继承Collection接口，数据结构是队列，特点是先进先出
